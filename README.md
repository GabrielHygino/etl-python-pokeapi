# ETL PokeApi

## Portuguese Tutorial

Esse é o meu projeto etl (extract, transform, load), onde eu pego todos os pokemons da pokeapi e os carrego em um banco de dados mongo ou em uma planilha.

Como utilizar?

Para rodar e conseguir em formato .csv basta rodar o arquivo etl-csv.
Já para conseguir os dados em uma collection do MongoDB basta rodar o arquivo etl-mongo.


## English Tutorial

This is my etl project (extract, transform, load), where I take all the pokemons from the pokeapi and load them into a mongo database or a spreadsheet.

How to use?

To run and get it in .csv format, just run the etl-csv file. 
To get the data in a MongoDB collection, just run the etl-mongo.
